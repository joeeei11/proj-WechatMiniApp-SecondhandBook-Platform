# proj-WechatMiniApp-SecondhandBook-Platform

**基于微信云开发的校园二手书交易小程序，支持发布、浏览、下单全流程。**

## 功能特性

- 书籍发布与浏览，支持图片上传和分类筛选
- 在线下单、确认、取消、完成订单全流程
- 微信云函数处理订单状态流转
- 用户个人中心与历史订单管理
- 消息通知与实时状态更新

## 快速开始

### 环境要求

- 微信开发者工具 >= 1.06
- 开通微信云开发环境

### 安装步骤

1. 克隆仓库

```bash
git clone https://github.com/joeeei11/proj-WechatMiniApp-SecondhandBook-Platform.git
```

2. 用微信开发者工具打开项目根目录，在 `app.js` 中填入云开发环境 ID
3. 在控制台一键部署所有云函数（`cloudfunctions/`）

### 基础用法

```
微信开发者工具 → 预览 / 真机调试
```
